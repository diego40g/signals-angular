NgFor vs @for - Renderizado de Listas

❌ Sintaxis Antigua (NgFor) - DEPRECADO


<ul>
  <li *ngFor="let user of users;
              let i = index;
              trackBy: trackByUserId">
    { { i + 1 } }. { { user.name } } ( { { user.age } } años )
  </li>
</ul>
  • {{ i + 1 }}. {{ user.name }} ({{ user.age }} años) {{ user.role }}

✅ Sintaxis Nueva (@for) - RECOMENDADO

<ul>
  @for (user of users; track user.id) {
    <li> { { $index + 1 } }. { { user.name } } ( { { user.age } } años )</li>
  } @empty {
    <li>No hay usuarios disponibles</li>
  }
</ul>
    @for (user of users; track user.id) {
  • {{ $index + 1 }}. {{ user.name }} ({{ user.age }} años) {{ user.role }}
    • } @empty {
  • No hay usuarios disponibles
    • }
