NgIf vs @if - Renderizado Condicional

❌ Sintaxis Antigua (NgIf) - DEPRECADO


<div *ngIf="showContent">
  <p>Contenido mostrado con *ngIf</p>
</div>

<div *ngIf="!showContent; else noContent">
  <p>Contenido visible</p>
</div>
<ng-template #noContent>
  <p>Contenido oculto</p>
</ng-template>

@if (showContent) {
  <p>Contenido mostrado con @if</p>
} @else {
  <p>Contenido oculto con @if</p>
}
